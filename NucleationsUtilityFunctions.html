Nucleation Functions Documentation

This documentation provides a full reference for the core functions used to simulate classical nucleation during metal 3D printing processes.

1. compute_undercooling(T_m, T)

Formula: ΔT = Tₘ − T

Description: Calculates undercooling, the key thermodynamic driving force for nucleation.

Returns: Undercooling in Kelvin (K).

2. compute_delta_Gv(L, delta_T, T_m)

Formula: ΔGᵥ = (L * ΔT) / Tₘ

Description: Computes the volumetric free energy difference between liquid and solid phases.

3. compute_critical_radius(gamma, delta_Gv)

Formula: r* = 2γ / ΔGᵥ

Description: Calculates the minimum stable nucleus size required for growth.

4. compute_energy_barrier_homogeneous(gamma, delta_Gv)

Formula: ΔG* = (16πγ³) / (3ΔGᵥ²)

Description: Returns the energy barrier for homogeneous nucleation.

5. compute_f_theta(theta_deg)

Formula: f(θ) = ((2 + cosθ)(1 - cosθ)²) / 4

Description: Calculates the geometric factor to reduce energy barrier in heterogeneous nucleation.

6. compute_energy_barrier_heterogeneous(delta_G_homo, f_theta)

Formula: ΔG*ₕₑₜ = f(θ) * ΔG*ₕₒₘₒ

Description: Returns the effective energy barrier considering surface effects or impurities.

7. compute_nucleation_probability(delta_G_star, T, k_B=1.38e-23)

Formula: P = exp(−ΔG* / kBT)

Description: Computes the probability of nucleation using Arrhenius-like Boltzmann statistics.

8. check_nucleation(probability)

Description: Performs a stochastic trial — returns True if a nucleus forms this timestep.

9. modify_gamma_by_impurity(gamma_0, C, alpha)

Formula: γ(C) = γ₀ * (1 - α * C)

Description: Modifies surface energy in presence of impurities to simulate heterogeneous sites.

10. modify_delta_Gv_field(base_delta_Gv, multiplier)

Description: Applies a field- or site-specific scaling to the driving force ΔGᵥ.

Designed for use in solidification modeling, 3D printing simulations, or kinetic Monte Carlo systems.
